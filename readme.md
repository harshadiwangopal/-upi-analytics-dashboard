# India UPI Payments Analytics (2021-2026)

## What is this project?
A data analytics dashboard analyzing 5 years of official NPCI UPI transaction data.

## Key Insights
1. UPI transaction volume grew 5x in 4 years (2021-2026)
2. Growth rate slowing from 77% to 29% — normal for maturing platforms
3. March is consistently the highest month due to financial year end activity
4. Average transaction value declining — UPI reaching everyday micro-transactions

## Tools Used
- Python
- Pandas
- Plotly
- Streamlit

## Data Source
NPCI Official UPI Product Statistics
https://www.npci.org.in/what-we-do/upi/product-statistics

## How to Run
pip install pandas plotly streamlit
